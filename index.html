My Projects

Order Management System

This software for a clothing business, developed using C# .NET and MySQL as the database, is software capable of adding, deleting, finding, or updating any order. It can also print the order as a PDF using Crystal Reports. The business can add new product categories and customer names to the database. The system has two additional sections: completed orders and pending orders. When an order is delivered to a customer, the user can enter the order ID in the completed section, and that order will automatically be removed from the pending orders table.

Employ Payroll Management System

This software is developed in Python utilizing the Tkinter library for frontend design. MySQL serves as the backend database. Additionally, an Admin panel and login system, which includes a forget password feature using an OTP system through emailid. Through the Admin panel, CRUD (Create, Read, Update, Delete) operations can be performed on employee data. Moreover, the software enables the calculation of salaries by incorporating various allowances such as DA (Dearness Allowance), HRA (House Rent Allowance), and generating a payslip, which is then mailed to the employee's email address

Career Guidance System

The software is developed in Python, utilizing the Tkinter library for frontend design, with MySQL serving as the backend database. It features a multiple-choice questionnaire (MCQ) designed to guide users in selecting the most suitable career options and colleges after completing their 12th grade education. The software analyzes the user's responses and provides recommendations accordingly.

Weather Forecast

This web-based responsive application is developed using HTML, CSS, and JavaScript, and it integrates with a weather API to provide real-time weather information. The app allows users to search for current weather conditions and forecast data for any city.
